# Map Icons

A collection of map marker icons for use in mapping applications and geographic information systems.

## Overview

This repository contains customizable icons designed for map markers, location pins, and geographic visualization.

## Installation

```bash
git clone https://github.com/katongole89/map-icons.git
```

## Usage

Icons can be used in web mapping libraries such as:
- Leaflet
- Mapbox
- Google Maps
- OpenLayers
